Project Description
3D object representation using various techniques and algorithms of Computer Graphics.
The project simulates the mirror reflection of 3D objects like Sphere, Cube, Cone, etc
This also applies transformation of objects and various Illumination models (Diffuse Reflection, Specular Reflection).
This is a part of B.E. Graphics Project done by Aashish, Anup, Bikash.
