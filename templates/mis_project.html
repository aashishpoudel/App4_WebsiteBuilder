Project Description

The project automates and analyzes the raw phone record details into different database tables, and
enables user, through C#.NET GUI, to inquire and generate different reports.
The report is presented in Grid form or Excel.
The project uses mySQL database as backend and C#.NET as frontend application. This is an undergrad Project done at Nepal Telecom by Aashish, Anjan, Anup, Bikash and Jeewan
