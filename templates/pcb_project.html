Project: PCB Milling Machine

The Project is comprised of Software, Embedded and Mechanical parts.
A PCB layout designed by some CAD PCB design software is converted into double line by Software.
The coordinates are fed to a microcontroller (AT89S51) which drives mechanical stepper motor via serial communication and using vector drawing technique.
The objective is to etch requested PCB Layout in double line format on a copper PCB plate.
This is an undergrad Embedded Project done by Aashish, Anjan, Anup, Bikash and Jeewan.
